# DESCRIPTION OF TASK

## Overview
A fully styled and interactive multi-page E-Learning Platform UI built using HTML, CSS, and JavaScript. The platform includes a home page with course listing, a student dashboard, a course detail page, and a lesson player page. It features 14 unique features — many of which are not found on any existing educational platform.

## What I Performed
- Designed and built a 4-page e-learning platform entirely from scratch using HTML, CSS, and vanilla JavaScript.
- Built a home page with a hero section, feature showcase, mood-based course suggestions, and a searchable, filterable course grid with 8 courses across 5 categories.
- Created a Student Dashboard with streak tracking, quick stats, mood picker, in-progress courses, weekly activity bar chart, subject analytics, quiz performance stats, activity heatmap, micro-badges system, and enrolled courses list.
- Built a Course Detail page with a dynamic progress circle, visual learning roadmap (nodes + connectors), full curriculum list, and course overview with topics covered.
- Built a Lesson Player page with YouTube video embedding, reading mode toggle, timestamp notes, AI study companion chat, focus mode, Pomodoro timer, and lesson navigation.
- Used localStorage to persist all data — progress, streak, badges, notes, mood — with no backend required.
- Split CSS into 4 separate files (main.css, dashboard.css, course.css, lesson.css) for clean organization.
- Made all 4 pages fully responsive for mobile, tablet, and desktop.

## Unique Features (Not on Existing Platforms)
1. **AI Study Companion** — Sidebar chat that answers questions about the current lesson topic
2. **Focus Mode** — One click removes all distractions, only video remains
3. **Timestamp Notes** — Pin notes at exact video moments and jump back to them
4. **Daily Learning Streak** — Tracks consecutive study days like Duolingo
5. **Mood-Based Suggestions** — Recommends course types based on how you feel today
6. **Micro-Badges System** — Earn badges at every milestone, not just course completion
7. **Personal Learning Analytics** — Weekly chart, subject bars, quiz stats, activity heatmap
8. **Pomodoro Study Timer** — Built-in 25/5 timer that integrates with the lesson player
9. **Visual Course Roadmap** — Node-based learning path showing completed, current, and locked lessons
10. **Reading Mode** — Toggle between video and text for every lesson

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, CSS Variables, Animations, Glassmorphism)
- Vanilla JavaScript (localStorage, IntersectionObserver, DOM manipulation)
- YouTube Embed API (for video embedding)
- Google Fonts (Inter, Space Grotesk)
- Font Awesome Icons

## How to Run
1. Clone the repository
2. Open `index.html` in any browser
3. Navigate using the navbar or click on any course card
4. No installation, no backend, no build step required
